01
OpsMesh // agent-native operations platform
+

Agent-native operations platform built on OpenClaw — the agent runtime Edward built from scratch. First product: Gig Radar — polls gig and job sources, normalizes and deduplicates listings, scores them via OpenClaw reasoning, fires Telegram alerts. OpenClaw owns the runtime; OpsMesh owns the domain logic, workflow state, and data model. Next: DM Shop Agent — conversational commerce over WhatsApp and Telegram.

OpenClaw Agent Runtime Telegram API WhatsApp Workflow State

architecture

OpenClaw provides the agent runtime and reasoning layer. OpsMesh implements domain-specific logic, persistent workflow state, and structured data models — cleanly separated from the runtime.

products

Gig Radar — live. DM Shop Agent — in development. Both built on the same OpenClaw foundation.

02
Shipd // deployment intelligence
+

Repo-aware deployment planning tool. Connects to GitHub read-only, scans for deployment signals — Dockerfiles, CI configs, env files, infra folders — and scores hosting platforms (Railway, Fly.io, Vercel, Render) against what it actually finds. Produces a saved plan with comparison view and evidence trail. No code writes. No deploy execution. Just clear decisions before you touch anything.

GitHub API Static Analysis Platform Scoring Decision Records

principle

Read-only. No writes. No deploys. Decision support only. Evidence-backed recommendations with full audit trail.

what it does

Reads structure. Detects deployment signals. Scores platforms by fit. Generates saved plans with traceable evidence.

03
MPP Studio // machine payments protocol console
+

Developer console for the Machine Payments Protocol. Register APIs, run sandbox 402 flows, inspect service contracts, graduate to live payment rails. Built for agent-to-agent commerce — where machines discover services, negotiate access, and settle payments without human intervention. HTTP 402 as first-class infrastructure, not just a status code.

MPP HTTP 402 Sandbox Service Contracts Agent Commerce

use case

Machines discovering services, negotiating terms, and settling payments without human intervention. The infrastructure layer of agent economies.

capabilities

API registration. Sandbox 402 flows. Contract inspection. Graduation path to live payment rails.
